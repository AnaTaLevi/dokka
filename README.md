# What Am I?

An api service which fetches addresses and distances between points

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install:

```bash
pip install json
pip install flask
pip install ast
pip install io
pip install math
pip install pandas
pip install requests
pip install itertools
pip install uuid
pip install sqlalchemy
```

## Usage

```bash
"/getAddresses"" 
Is a HTTP POST method that enables to upload a
CSV file with point locations ( LAT/LON ) 

"/getResult"
Is an API to retrieve results of “/api/getAddresses” API
identified by “result_id”.
```
